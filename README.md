# FreeRTOS on ESP32

Hands-on experiments and examples exploring FreeRTOS concepts on ESP32 using the Arduino IDE framework.

## Topics

1. Task Scheduling
2. Memory Management
3. Queue
4. Mutex
5. Semaphore
6. Software Timers
7. Interrupts
8. Deadlock and Starvation
9. Priority Inversion
10. Multicore Systems

## Development Environment

- Board: ESP32
- Framework: Arduino
- IDE: Arduino IDE
- RTOS: FreeRTOS (built into the ESP32 Arduino core)
